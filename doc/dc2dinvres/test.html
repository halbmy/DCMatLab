DC2DInvRes - Download Test Version(v.2.4b1 18.10.2004)

Note: The test version has full facilities excepting model export and export of eps files. Additionally, the model is displayed with text overlay and some lines.

System requirements: Windows 98/Me/2000/XP
It is recommended to have at least 256MB memory to compute bigger data files.


Installation guide:
  1. Download the Setup program
  2. Execute it and follow the instructions, it will create Start Menu Folder and Desktop Icons, it will create Start Menu Folder and Desktop Icons
  3. The program is written in MatLab, so most users will need additional libraries provided in the zip file system.zip
  4. Download it and unpack all files into a system path (e.g. C:\Windows\system32), which needs administration rights, or into the program directory (usually C:\Programme\DC2DInvRes).
