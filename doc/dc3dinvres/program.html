DC3DInvRes - Download Program(v.2.7 23.06.2005)

System requirements: Windows 98/Me/2000/XP
It is recommended to have at least 256MB memory to compute bigger data files of more than thousand data points.


Installation guide:
  1. Download the Setup program
  2. Execute it and follow the instructions, it will create Start Menu Folder and Desktop Icons, it will create Start Menu Folder and Desktop Icons
  3. The program is written in MatLab (v6.5), so most users will additionally need the libraries provided in the zip file system.zip
  4. Download it and unpack all files into a system path (e.g. C:\Windows\system32), which needs administration rights, or into the program directory (usually C:\Programme\DC3DInvRes).

Note: Until no valid license file is present, the program is running as test version. The test version has full facilities excepting model export, export of eps files and workspace saving. Additionally, the model is displayed with text overlay and some lines to prevent screen captures.

A license file (dc3dinvres.lic) can be obtained by the author and is restricted to a computer. Changes in the hardware may need a license renewal. Please install dc3dinvres, start the file startlic.exe (in the program directory) and send the code via email.
