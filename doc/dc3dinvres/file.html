DC3DInvRes - File Menu

Open data file

There are three supported data formats, which will be detected automatically:

  1. an electrode number based format (the # character is used for comments):
     number_of_electrodes
     x_1 y_1 z_1 # position of first electrode (z optional)
     ...
     x_n z_n # position of last electrode
     number_of_data
     a_1 b_1 m_1 n_1 rhoa_1 (error_1) # first datum point
     ...
     a_k b_k m_k n_k rhoa_k (error_k) # last datum point (errors optional)
    where a_i, b_i ... are the numbers (not positions) of used electrodes in the list above. Infinite electrodes for b and/or n are defined by number 0. 
  2. Absolute electrode positions (surface data only)
     x_a y_a x_b y_b x_m y_m x_n y_n rhoa
     ...
  3. RES3DINV data format (see res3dinv homepage for datails)

Note that the electrode positions are arbitrary and independent on any parameterization or grid.

Hotkey for Open data file: Shift+O

Read pro file

Many 3d data sets are aquired in form of 2d profiles, either parallel and perpendicular or not. Then a file defining the profile locations can be read in, which looks as follows:

filename1 x_start y_start x_end y_end
filename2 ...

The 2d-files are in the res2dinv or the above descibed format except that y positions of electrodes are left out.
This form of data input is recommended if profiles are not parallel to x- or y-direction, so that they can be plottet in Display=>Data as profile data.

Hotkey for Read pro file: Shift+P

Add Data

The data read from the file are added to the existing data. So it is easy to combine different configurations on the same profile to get better results.

Hotkey for Add data: Shift+A

Save Workspace

The complete workspace (e.g. data, model and sensitivities) are saved to a *.mat file, defaulting the same root like the data file with extension MAT. When loading again, the program all variables including options are recovered. For saving inversion results it is recommended to Clear Sensitivity first to save disk space.

Hotkey for Save Workspace: Shift+S

Load Workspace

A former saved mat-file can be loaded to work on with the model.
Attention: files named *-sens.mat or *-s.mat are saved sensitivities and contain no model!

Hotkey for Load Workspace: Shift+L

Export Figures

All active windows are exported to graphics files. Filename(s) and path are determined by a Explorer menu.
Possible file formats are png (indexed bitmap) or eps (vector graphics), which can be transformed to pdf easily using epstopdf.
It is recommended to resize the Figure windows before exporting such that subfigures are not overlapping.

Hotkey for Export Figures: Shift+E

Export Colorbar

The colorbar displayed in the main GUI can also be exported to graphics file.

Exit

Termination of program. All windows are closed.

Hotkey for Exit Program: Shift+X
